ignoreSecondDecimalInInput

  • ignoreSecondDecimalInInput(eventObject: KeyboardEvent<HTMLInputElement>, currentValue: string | number): void
    • Defined in dom/index.ts:349

    In safari browser, you can input multiple decimals. This method can be used to prevent that.

    remarks

    It's strongly recommended to use this method on onKeyDown event to prevent the key from registering

    example
    <input
      type="number"
      onInput={this.handleInput}
      value={this.state.value}
      onKeyDown={(eventObject) => ignoreSecondDecimalInInput(eventObject, this.state.value)}
    />

    Parameters

    • eventObject: KeyboardEvent<HTMLInputElement>

      onKeyDown event object

    • currentValue: string | number

      Current value of input field

    Returns void

smoothScrollToTop

  • smoothScrollToTop(): void
    • Defined in dom/index.ts:315

    This function scrolls the page to the top. If window.scroll is available and works perfectly, this function uses the smooth scroll behaviour of window and scrolls with ease in animation. Else, It directly scrolls to top without animation in case of error with window.scroll.

    without animation if window object is not present.

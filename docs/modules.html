postWindowMessage

  • postWindowMessage(targetWindow?: Window, action?: string, params?: Object, targetOrigin?: string): void
    • Defined in dom/index.ts:346

    This method is used to push a message along with some data to a window instance. Useful in scenarios like iFrames, Webviews or Window Modals.

    remarks


    It is advisable by MDN to always send targetOrigin (usually your host) to avoid security breach.
    When writing the listener of this message, ensure event.origin is verified and acknowledged before further processing.

    example
    const newWindow = window.open("https://groww.in/random-route", "_blank");

    postWindowMessage(newWindow, 'CHANGE_THEME', { theme: 'dark' }, "https://groww.in");

    Parameters

    • targetWindow: Window = window

      Target window that needs to listen to the message. Defaults to current window.

    • action: string = 'WINDOW_ACTION'

      Action type

    • params: Object = {}

      POJO - Any payload to be passed along with the action

    • targetOrigin: string = ''

      Target origin identifier. This is to verify at the receiver end, if origin is as expected or not. Usually defaults to the host.

    Returns void

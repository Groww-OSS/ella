dispatchCustomEvent

  • dispatchCustomEvent(eventName: string, eventDetails?: object): void
    • Defined in dom/index.ts:50

    This method can be used to dispatch any custom event.

    remarks

    It's strongly recommended to add the event name in utils/constants/CUSTOM_EVENTS and use. No literals allowed.

    example
    dispatchCustomEvent(CUSTOM_EVENTS.file_loaded)
    dispatchCustomEvent(CUSTOM_EVENTS.file_loaded, {user: 'Ella'})

    Parameters

    • eventName: string

      Name of the event that you want to dispatch

    • eventDetails: object = {}

      OPTIONAL param, eventDetails object can be used to pass extra details

    Returns void
